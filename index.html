Hi there!

Welcome to the paint app. Here you can draw and paint to your heart's desire.
This browser-based app is pretty straight forward.

👈 There is a menu to your left. Below are general guidelines for using it.

  • Ink Pen 🖊 - Draw a solid line
  • Paint Brush 🖌 - Paint a tapered line
  • Pencil ✏️ - Draw a pencil line
  • Soap 🧼 - Wash away those mistakes
  • Fill Bucket 🪣 - Fill background with a solid color
  • Rainbow Effect 🌈 - Activate every color in the spectrum
  • Screen Effect 🌖 - Create lighter effect where colors overlap
  • Multiply Effect 🌒 - Create darker effect where colors overlap
  • Color Picker ⬛ - Choose color to use
  • Clear Canvas 💀 - Scrap everything and start over

To change the brush size, press and hold the bracket keys.
Closed bracket ( ] ) to make it larger and open bracket ( [ ) to make it smaller.

If you forget something, you can always hover over the emoji icons in the menu.
